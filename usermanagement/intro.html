Usermanagement¶

The user management component has the following functionality:

  • store users with profiles in a MongoDB storage
  • login users
  • provide access tokens for users via OAuth2

Roadmap¶

  • Version 1
    • Stories
      • store a fixed set of users in a dictionary
      • User login
      • allow OAuth clients to obtain access tokens for a user
      • allow OAuth clients to obtain profile information for users using the access token
